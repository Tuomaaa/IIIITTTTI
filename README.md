## 技术栈

- React + Vite
- Tailwind CSS
- html2canvas（分享图生成）

## 本地开发

```bash
npm install
npm run dev
```

## 部署到 GitHub Pages

1. 创建 GitHub repo，push 代码
2. 进入 repo → Settings → Pages → Source 选 **GitHub Actions**
3. push 到 `main` 分支会自动部署
